# 信用卡現金回饋計算器

一個簡單易用的網頁工具，幫助您快速找到消費項目中回饋最高的信用卡。

## 功能特色

- 🔍 **智能搜索**：輸入商家或項目名稱，系統自動匹配
- 💰 **精準計算**：考慮回饋率、消費上限等因素
- 📊 **結果排序**：按回饋金額高低排序，最優選擇一目了然
- 🎯 **快速選擇**：提供常用消費項目的快速按鈕
- 📱 **響應式設計**：支援桌面和移動設備

## 使用方法

1. 開啟 `index.html` 檔案
2. 輸入消費的商家或項目名稱
3. 輸入消費金額
4. 點擊「計算回饋」按鈕
5. 查看各張卡片的回饋結果

## 自訂信用卡資料

您可以編輯 `cards.json` 檔案來：

- 新增或移除信用卡
- 修改回饋率和消費上限
- 更新支援的商家列表

### cards.json 檔案格式

```json
{
  "cards": [
    {
      "id": "card-id",
      "name": "卡片名稱",
      "cashbackRates": [
        {
          "rate": 3.5,
          "cap": 20000,
          "items": ["商家1", "商家2", "..."]
        }
      ]
    }
  ]
}
```
