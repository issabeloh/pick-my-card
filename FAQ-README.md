# FAQ 頁面使用說明

## 📋 概述

本專案已建置獨立的 FAQ（常見問題）頁面，使用手風琴式設計，方便使用者查看問題和答案。

## 🗂️ 檔案結構

- **faq.html** - FAQ 頁面主體
- **faq.css** - FAQ 頁面專用樣式
- **faq.js** - FAQ 資料載入和互動邏輯
- **cards.data** - FAQ 資料來源（JSON 格式）

## 📊 Google Sheet 資料結構

在 Google Sheet 中建立名為「FAQ」的工作表，欄位如下：

| 欄位名稱 | 說明 | 範例 | 必填 |
|---------|------|------|------|
| **id** | 問題編號（唯一識別） | "1", "2", "3" | ✅ |
| **category** | 分類名稱 | "使用相關", "信用卡相關", "帳號相關" | ✅ |
| **question** | 問題內容 | "如何開始使用本工具？" | ✅ |
| **answer** | 答案內容（HTML 格式） | 見下方範例 | ✅ |
| **order** | 顯示順序（數字越小越前面） | "1", "2", "3" | ✅ |
| **isActive** | 是否啟用 | "TRUE" 或 "FALSE" | ✅ |

## ✍️ 答案格式範例


## 🔄 更新 FAQ 資料流程

### 步驟 1: 在 Google Sheet 編輯資料
1. 開啟您的 Google Sheet「FAQ」工作表
2. 新增或編輯問題
3. 確保 `isActive` 欄位設為 `TRUE`

### 步驟 2: 匯出為 JSON
1. 在 Google Sheet 中執行匯出腳本
2. 將 FAQ 資料匯出為 JSON 格式

預期的 JSON 格式：
```json
{
  "cards": [ ... ],
  "payments": [ ... ],
  "faq": [
    {
      "id": "1",
      "category": "使用相關",
      "question": "如何開始使用？",
      "answer": "<p>請點擊首頁的「開始使用」按鈕。</p>",
      "order": 1,
      "isActive": true
    },
    {
      "id": "2",
      "category": "信用卡相關",
      "question": "支援哪些信用卡？",
      "answer": "<p>目前支援 <strong>13 張台灣主流信用卡</strong>。</p>",
      "order": 2,
      "isActive": true
    }
  ]
}
```

### 步驟 3: 更新 cards.data
1. 將新的 JSON 資料整合到 `cards.data` 檔案中
2. 確保 JSON 格式正確
3. 上傳到 GitHub

### 步驟 4: 測試
1. 開啟 `https://pickmycard.app/faq.html`
2. 檢查 FAQ 是否正確顯示
3. 測試分類篩選功能
4. 測試手風琴展開/收合

## 🎨 頁面功能

### 1. 手風琴式設計
- 點擊問題即可展開/收合答案
- 流暢的動畫效果
- 圖示旋轉提示展開狀態

### 2. 分類篩選
- 自動根據資料生成分類按鈕
- 點擊分類可篩選相關問題
- 「全部」按鈕顯示所有問題

### 3. 響應式設計
- 支援桌面、平板、手機顯示
- 自適應排版
- 觸控友善

### 4. 導航功能
- 首頁 → FAQ：點擊 footer 的「常見問題 FAQ」連結
- FAQ → 首頁：點擊 FAQ 頁面右上角的「返回首頁」按鈕

## 🔧 維護建議

### 定期更新
- 每月檢查 FAQ 內容是否需要更新
- 根據使用者反饋新增常見問題
- 移除過時或不再相關的問題

### 內容品質
- 答案保持在 100-250 字
- 使用清晰簡潔的語言
- 適當使用粗體和列點提升可讀性

### 分類管理
- 建議分類數量：3-6 個
- 分類名稱簡短明確
- 每個分類至少包含 2-3 個問題

## ❓ 常見問題

### Q: 如果沒有 FAQ 資料會怎樣？
A: 頁面會顯示「FAQ 內容即將推出，敬請期待！」的訊息。

### Q: 可以使用 Markdown 格式嗎？
A: 不行，答案必須使用 HTML 格式。建議在 Google Sheet 直接寫 HTML。

### Q: 如何隱藏某個問題？
A: 將該問題的 `isActive` 欄位設為 `FALSE` 即可。

### Q: 如何調整問題順序？
A: 修改 `order` 欄位的數字，數字越小越前面。

### Q: 可以在答案中加入圖片嗎？
A: 可以，使用 HTML `<img>` 標籤：
```html
<img src="圖片網址" alt="描述" style="max-width: 100%;">
```
